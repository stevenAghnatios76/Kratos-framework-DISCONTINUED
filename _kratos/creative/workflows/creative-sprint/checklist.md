# Creative Sprint Checklist
- [ ] Creative challenge clearly defined with problem statement
- [ ] Constraints and success criteria documented
- [ ] Empathize phase completed with user insights and personas
- [ ] Problem definition anchored in user research
- [ ] Solve phase produced ranked solution proposals
- [ ] Innovation phase identified disruption opportunities
- [ ] All three phases synthesized into unified creative brief
- [ ] Solutions mapped to user needs from empathy phase
- [ ] Implementation priorities and next steps defined
