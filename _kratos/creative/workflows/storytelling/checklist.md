# Storytelling Checklist

## Audience and Purpose
- [ ] Target audience clearly defined
- [ ] Desired emotional response identified
- [ ] Core message distilled to one sentence

## Framework
- [ ] Story framework selected with rationale
- [ ] Framework structure applied to content

## Story Arc
- [ ] Opening hook grabs attention in 3 seconds
- [ ] Rising tension builds toward climax
- [ ] Transformation arc is clear
- [ ] Resolution delivers the core message

## Emotional Design
- [ ] Emotional beats mapped
- [ ] Each beat serves the message
- [ ] Tension and relief balanced

## Polish
- [ ] Language refined — every word earns its place
- [ ] Story reads aloud naturally
- [ ] Authentic — no fabricated emotional beats
