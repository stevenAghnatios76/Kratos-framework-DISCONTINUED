## Context
- [ ] Audience defined (investors, partners, internal)
- [ ] Funding stage or purpose identified
- [ ] Key ask is specific and justified
## Structure
- [ ] All standard slides present: Problem, Solution, Market, Model, Traction, Team, Ask
- [ ] Each slide has one key message
## Data
- [ ] Market size data is sourced and credible
- [ ] Traction metrics are specific
- [ ] Financial projections are realistic
## Narrative
- [ ] Story flows from problem to opportunity
- [ ] Opening hook is compelling
- [ ] Closing call to action is clear
## Output Verification
- [ ] Pitch deck spec generated at {creative_artifacts}/
