# Step 3: Technique Execution

1. Run the selected technique step by step
2. For each round:
   - Generate 5-10 ideas based on the technique's methodology
   - Present ideas to user for reaction
   - Build on user feedback to generate more ideas
   - Capture all ideas without filtering
3. If using multiple techniques, run each sequentially
4. Aim for 15-30 total ideas before moving to organization
5. Encourage quantity over quality at this stage
