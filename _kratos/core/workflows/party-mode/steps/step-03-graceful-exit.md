# Step 3: Graceful Exit

1. Kratos summarizes the key takeaways from the discussion
2. List action items that emerged, attributed to the relevant agent
3. Ask user if they want to:
   - Save the discussion transcript
   - Activate a specific agent to follow up on a topic
   - Start a workflow based on a discussed idea
4. If saving: write transcript to `{creative_artifacts}/party-mode-{date}.md`
5. Thank the participants and close the session
