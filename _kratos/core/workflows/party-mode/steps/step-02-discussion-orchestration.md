# Step 2: Discussion Orchestration

1. Kratos acts as moderator for the discussion
2. Present the topic to all agents
3. For each round:
   - Have 2-3 agents respond in character based on their persona
   - Each agent speaks from their domain expertise
   - Agents may agree, disagree, or build on each other's points
   - Kratos summarizes key points after each round
4. After 3-4 rounds, check with user:
   - Continue discussion?
   - Change topic?
   - Ask a specific agent to elaborate?
   - Wrap up?
5. Keep each agent response to 2-3 paragraphs maximum
6. Maintain distinct voices — each agent should sound different
